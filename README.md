# XAUT_Doctoral_Dissertation_Template
本项目为西安理工大学博士学位论文模板，仅供学术参考和个人使用。本模版为非官方模板，全文参考《西安理工大学研究生学位论文撰写规范》完成。

本模板在武汉大学黄正华老师所提供的《武汉大学学位论文模板》以及内蒙古科技大学李琦老师所提供的《内蒙古科技大学学位论文模板》基础上修改完成。衷心感谢黄正华老师(<http://aff.whu.edu.cn/huangzh/>)和李琦老师(richey@imust.edu.cn)前期所做的卓越工作，使得本模板的设计与排版工作能够顺利进行。此外，感谢西安理工大学博士杜森和硕士陈振荣的学位论文以供本模版纠偏。珠玉在前，感谢前人的工作！

再次强调，本模板为非官方版本，全文参考《西安理工大学研究生学位论文撰写规范》完成，仅供学术参考和个人使用。使用本模板所产生的排版效果及其他后果与原学校及原作者无关。用户应根据实际要求自行检查与修改，以确保符合相关学位论文规范。
